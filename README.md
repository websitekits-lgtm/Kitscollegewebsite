# Krishna Chaitanya Institute of Technology & Sciences

Website for Krishna Chaitanya Institute of Technology & Sciences (KITS), Markapur.

## Tech stack
- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- React Router
- GSAP, Three.js (react-three-fiber)

## Local development

```sh
npm install
npm run dev
```

Then open http://localhost:8080

## Build

```sh
npm run build
npm run preview
```
